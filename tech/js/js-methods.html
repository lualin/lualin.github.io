Javascript Methods List

Javascript Methods - Basic built-in objects

1) Object object

Top-level object of javascript
Construct

var object={};
var object=new Object();

- Methods of the Object object

constructor() : Represents the object's constructor function.
hasOwnProperty(name) : checks whether an object has a name property
isPrototypeof(object): Checks whether an object is the prototype of object.
propertyIsEnumerable(name) : Check whether enumerable with loop statement
toLocaleString() : Converts an object to a string in the language appropriate to the host environment.
toString() : convert object to string
valueOf() : display the value of an object

2) Number object

Objects used to represent numbers
Produce

var num = 1;
var num = new Number(1);

- Methods of the Number object

toExponential() : Returns a string representing a number in exponential notation // Number of significant digits
toFixed() : Returns a string representing a number in fixed decimal notation // number of decimal places
toPrecision() : Returns a string representing a number in exponential or fixed-point representation depending on its length.

- Properties of the Number constructor function

MAX_VALUE: The maximum number that can be expressed in JavaScript
MIN_VALUE: The minimum number that can be expressed in JavaScript
NaN: Not a Number
POSITIVE_INFINITY : positive infinity number
NEGATIVE_INFINITY : negative infinity number

3) String object

An object used to represent a string

- Properties of the String object

length: Displays the length of the string

- Methods of the String object

charAt(position) : return the character at that index
charCodeAt(position) : Returns the corresponding index character as Unicode
concat(args): concatenates strings entered as parameters
indexOf(searchString, position): returns the index of the matching string starting from the beginning
lastIndexOf(searchString, position) : returns the index of the matching string starting from the back
match(regExp) : check if regExp exists in string
replace(regExp, replacement) : Replace regExp in string with replacement and return
search(regExp): Returns the location of a string matching regExp
slice(start, end): returns a string at a specific position
split(separator, limit): Returns an array after cutting the string with separator
substr(start, count) : Cuts the string from start to count and returns it
substring(start, end): cuts the string from start to end and returns it
toLowerCase() : Converts a string to lower case and returns it.
toUpperCase() : Converts a string to uppercase and returns it.

- HTML-related methods of the String object

anchor() : add a tag to string
big() : add big tag
blink() : add blink tag
bold() : add b tag
fixed() : add tt tag
fontcolor(colorString) : add font tag and color attribute
fontsize(fontSize) : add font tag and size attribute
italics() : add i tag
link(linkRef) : add a tag and href attribute
small() : add small tag
strike() : add strike tag
sub() : add sub tag
sup() : add sup tag

Method chaining is possible because String objects return the contents of the string without altering it.

4) Array object

An object used to represent an array
Produce

var array=[value1, value2...]; // initialization upon creation
var array=new Array(); // just create an array
var array=new Array(array size); // create space at the same time as creation
var array=new Array(value1, value2); // initialization upon creation

- Properties of Array object

length: returns the size of the array

- Methods of the Array constructor function

Array.isArray() : check if it is an array

- Array object methods

Methods that change properties
pop() : Removes the last element of an array and returns it.
push() : Adds a new element to the end of an array
reverse() : reverse the order of elements in an array
sort() : Sort array elements // default string ascending order
  *Sort in descending order: array.sort((a,b)=>{return b-a});
  *Shuffle: arr.sort(()=>{return 0.5-Math.random()});
splice() : Removes the specified part of an element and returns the deleted element

Methods that don't change properties
slice() : Returns a specified portion of an element
concat(): Creates an array by combining all the elements of the array entered as parameters and returns it
join() : Returns after converting all elements in the array to a string
indexOf() : Search the position of a specific element from the beginning of the array
lastIndexOf(): Search the position of a specific element from the back of the array

Iteration method
forEach() : Arrays can be used like a for in loop
map() : Creates a new array by applying certain rules to an existing array.

Conditional method
filter() : Creates a new array by extracting elements that satisfy certain conditions
every() : Checks if an element of an array satisfies a condition
some() : Checks if the elements of an array satisfy at least one condition

Operation method
reduce() : Executes a function that groups the elements of the array by two from the left until there are only one element in the array.
reduceRight() : same as above, but run from the right
Array.//prototype//.remove =function (i) { this.splice(i, 1); } // Add remove function to prototype
arr.remove(i); // remove that index value

5) Date object

An object that displays the date and time
Produce

var date = new Date(); // If no parameters are entered, reset to the current time

Parameter example
String: new Date('December 11, 2018');
Numbers: new Date(2018, 12, 11, 2, 24, 23); // year, month-1, day, hour, minute, second, millisecond order
Unix time: new Date(1351511); // elapsed time in milliseconds since 12:00 midnight on January 1, 1970

Methods of the Date object
getTime() : Returns Unix time, used to calculate date interval
*There are a lot of methods, but most of them consist of get and set, and the function can be inferred by the method name.
*The getYear() method returns different results depending on the browser, so use getFullYear()

6) Math object

Objects with properties and methods related to mathematics
The Math object is used as a variable // Math.property, not a function
It is the only built-in object that does not use a constructor

- Methods of the Math object
abs() : returns absolute value
acos() : Returns the arc cosine value
asin() : Returns the arc sine value
atan() : Returns the arc tangent value
atan2(y, x) : Returns the arc tangent of the ratio of x and y
ceil() : Returns the smallest integer greater than or equal to
cos() : returns the cosine value
exp() : Returns the square of the natural logarithm
floor() : Returns the largest integer less than or equal to
log() : return log value
max() : returns the largest value among parameters
min() : Returns the smallest value among parameters
pow(x, y) : returns x raised to the power of y
random() : Returns a random number from 0 to 1
round(): rounds up and returns
sin() : returns the sine value
sqrt() : Returns the square root
tan() : Returns the tangent value

7) JSON (Javascript Object Notation) object

A string in the form of a JavaScript object
Methods of the JSON object

JSON.stringify() : Convert a JavaScript object to a JSON string
* You can check the entire object property as a string rather than an object, etc.

JSON.parse() : Convert JSON string to JavaScript object
toJSON: function() {return object}; : If there is a toJSON method in the object, only the return value of that method is returned; otherwise, all keys and properties are returned.
